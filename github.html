Github tutorial

How to get an extension installation package

  • Click the "GitHub" button next to the extension listed on the Home page.
  • To download the latest release follow the link labeled as "Latest" in the "Releases" block on the right.
  • In the "Assets" block, select the download link that looks like "pkg_extensionname.zip".

Terms

If you’d like to report bugs in our extensions:

  • Click the "GitHub" button next to the extension listed on the Home page.
  • Go to the "Issues" page of this extension (item in the top menu).
  • Create an issue (green button "New issue" in the top right corner).

If you’d like to fix bugs in our extensions:

  • Click the "GitHub" button next to the extension listed on the Home page.
  • Go to the "Pull requests" page of this extension (item in the top menu).
  • Create a pull request (green button "New pull request" in the top right corner).
  • Your pull request will then be reviewed.

Please mind that custom changes not relevant for all users cannot be accepted.
